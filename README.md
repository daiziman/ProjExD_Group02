# DOOM風FPSゲーム

MiniPy3DR(pygame製の3Dレンダラ)のFPSツールキット `minipy3dr.fps` 
を使ったDOOM風FPSゲーム

## 実行環境の必要条件

* Python >= 3.10
* pygame >= 2.5
* minipy3dr >= 0.5.0 (`pip install minipy3dr`)

## 実行方法

```
python main.py
```

## ゲームの概要
* 爆発する樽や弾を防ぐ遮蔽物などのギミックを活かしつつ，敵を全滅させればクリア，体力が尽きればゲームオーバーとなる

## ゲームの遊び方
* `WASD`：移動 / マウス：視点操作 / `Shift`：ダッシュ
* 左クリック：射撃 / `Space`：ジャンプ / `Q`・`E`：左右へのリーン（覗き込み）/ `ESC`：終了
* 赤い帯の樽は撃つと爆発し，周囲の敵やプレイヤーにもダメージを与える（近くの樽に誘爆することもある）
* 緑色のクレートで体力を回復，青色のクレートで弾薬を補給できる
* 遮蔽物（コンテナ）は弾を防ぐが，リーンやジャンプを使うと越しに攻撃・移動できることがある
* 敵を全滅させるとゲームクリア，体力が0になるとゲームオーバー

## ゲームの実装
### 共通基本機能
* マップデータ（`game/level.py`）に基づく壁・床・天井の自動生成と装飾（幅木，天井の飾り帯，天井パイプ，行き止まりの瓦礫，明滅する天井ランプ）
* 敵AIによる追跡・攻撃，武器の発射，アイテム（体力・弾薬）の取得，HUD（体力・弾薬バー，スコア，撃破数）表示，イベントに応じた効果音再生（`game/features/` 配下の各`Feature`）
* 照準（クロスヘア）とヒット時の演出，開始時のミッション表示，低体力時の画面演出，勝敗画面の描画

### 分担追加機能
* 効果音追加（担当：笹本）:銃声音などを付けた
* 背景音追加（担当：笹本）:後ろで流れるBGM追加した
* 色変更（担当：笹本）:床・壁の色を変更した
* ジャンプ＆リーン機能（担当：C0C25006）：`game/core.py`。`Space`キーで重力に従い自然に跳躍・着地するジャンプ動作と，`Q`/`E`キーで体を左右に傾けて遮蔽物越しに覗き込むリーン動作を追加。ジャンプ中は一定の高さ以下の障害物（樽・コンテナ）を飛び越えられる

* クールダウン機能（担当：あまり）：クリックした分だけ弾が打てるのを制限する機能
* 連射機能（担当：あまり）：マウス左クリックを長押しすると連射する機能
* リロード機能（担当：あまり）：rキーを押すとリロードする機能
* ショットガン機能（担当：あまり）：1クリックで複数の弾がでて散弾する機能
* 武器切替機能（担当：あまり）:キー1,2,3を押すと武器が切り替わる機能
* 反動とフラッシュエフェクト（担当：あまり）:撃った瞬間に反動とフラッシュが現れる機能
### メモ
* ゲームエンジン本体は `minipy3dr.fps.FPSGame` を継承した `Game` クラス（`game/core.py`）で構成
