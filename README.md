# Magical Shooting

## 実行環境の必要条件

* Python >= 3.10
* pygame >= 2.5.2
* minipy3dr >= 0.5.0

必要なライブラリは、次のコマンドでインストールできます。

```bash
python -m pip install "pygame>=2.5" "minipy3dr>=0.5.0"
```

conda環境では(推奨機) pathが通ってない場合 

```bash
C:\Users\Admin\anaconda3\envs\ProjExD\python.exe -m pip install "pygame>=2.5" "minipy3dr>=0.5.0"
```

プロジェクトのフォルダで、次のコマンドを実行するとゲームが始まります。

```bash
python magical_shooting.py
```

## ゲームの概要

* `minipy3dr.fps`を利用して制作した、DOOM風の3D FPSゲーム
* プレイヤーは廃墟のようなマップを探索し、アイテムや地形のギミックを活用しながら敵と戦う
* 敵を累計12体倒すとゲームクリアとなり、プレイヤーのHPが0になるとゲームオーバーとなる
* 敵はマップ内を巡回し、プレイヤーを発見すると追跡や近距離・遠距離攻撃を行う

## ゲームの遊び方

* `W`、`A`、`S`、`D`キーで移動し、マウスで視点を操作する
* `Shift`キーでダッシュし、`Space`キーでジャンプする
* マウスの左ボタンで射撃する。左ボタンを押し続けると連射できる
* `Q`、`E`キーで左右にリーンし、遮蔽物から覗き込む
* `R`キーでリロードする
* `1`、`2`、`3`キーで、ピストル、ライフル、ショットガンを切り替える
* 鍵を取得した後、扉の近くで`E`キーを押すと扉を開けられる
* `Esc`キーでゲームを終了する
* 敵を倒すほか、ゴールドアイテムやレアアイテムを取得するとスコアが増える
* 赤色のパワーアップアイテムを取得すると、30秒間スコアが2倍になり、弾薬が自動回復する
* 赤い帯の樽を撃つと爆発し、周囲の敵やプレイヤーにダメージを与える。近くの樽に誘爆することもある
* 緑色のアイテムでHPを25回復し、青色のアイテムで弾薬を15補給できる
* 敵を累計12体倒すとゲームクリア、HPが0になるとゲームオーバーとなる

## ゲームの実装

### 共通基本機能

* マップデータ（`game/level.py`）に基づく壁、床、天井の自動生成
* 幅木、天井の飾り帯、天井パイプ、行き止まりの瓦礫、明滅する天井ランプの描画
* 爆発樽と遮蔽物の配置、当たり判定、爆発による範囲ダメージ
* 照準とヒットマーカー、開始時のミッション表示、低HP時の画面演出、勝敗画面の描画
* HP、弾薬、スコア、撃破数などのゲーム進行管理

### 分担追加機能

* アイテム機能（担当：吉荒）
  * 回復、弾薬、ゴールド、レア、鍵、パワーアップアイテムの配置
  * アイテムの回転・上下移動アニメーションと、取得時のパーティクル・フラッシュ演出
  * ゴールドアイテムによる100点、レアアイテムによる500点のスコア加算
  * 鍵を取得して扉を開く進行ギミック
  * 30秒間のスコア2倍と弾薬自動回復を行うパワーアップ

* HUD機能（担当：高橋）
  * HP残量に応じたHPバーの色変更と点滅
  * 撃破ログ、スコア、撃破数の表示
  * プレイヤーとマップの位置関係を示すミニマップ
  * 照準を合わせた敵の名前と残りHPの表示

* サウンド・配色機能（担当：笹本）
  * 射撃、命中、敵の撃破、アイテム取得、勝敗などに対応した効果音
  * 通常時、低HP時、ゲームクリア時、ゲームオーバー時のBGM切り替え
  * 床と壁の配色変更

* ジャンプ・リーン機能（担当：安藤）
  * `Space`キーによる、重力に従ったジャンプと着地
  * `Q`、`E`キーによる左右へのリーン
  * ジャンプ中に一定の高さ以下の障害物を飛び越えるための判定

* 武器機能（担当：甘利）
  * 射撃クールダウンとマウス長押しによる連射
  * 武器ごとに異なる装弾数とリロード時間
  * ピストル、ライフル、ショットガンの切り替え
  * ショットガンの散弾、射撃時の反動、マズルフラッシュ

* 敵機能（担当：中野）
  * HP、移動速度、得点などが異なる複数種類の敵
  * 被弾した敵を一瞬白くするフラッシュ演出
  * 巡回と追跡の状態遷移、近距離攻撃、遠距離攻撃
  * 敵を倒した5秒後に別の場所へ出現させるリスポーン機能
